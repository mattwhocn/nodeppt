#nodeppt的使用

## install

```sh
   npm install -g nodeppt
```

## create ppt

```sh
   nodeppt create [ppt-name]
```

## config

```sh
   title: 这是演讲的题目
   speaker: 演讲者名字
   url: 可以设置链接
   transition: 转场效果，例如：zoomin
   files: 引入js和css的地址，如果有的话~自动放在页面底部
```

## start

```sh
   nodeppt start -p <未被使用的端口>
```
or
```sh
   nodeppt start -d <ppt所在目录>
```
